## Geting started
The script that checks if there are open pull requests for a repository. An url like https://github.com/USER/REPO must be passed to the script
## Instalation
```sh
   chmod +x git_api.sh
```

## Requirements
* Linux OS
* bash
* curl
* qj

## Usage
```sh
   ./git_api.sh https://github.com/torvalds/linux
```
